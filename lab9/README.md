# Simple Note dApp

Децентрализованное приложение для хранения заметок в блокчейне BNB Smart Chain.

## Файлы:
- `index.html` - главная страница
- `style.css` - стили
- `script.js` - подключение к смарт-контракту
- `Note1.sol` - смарт-контракт

## Как использовать:
1. Разверните Note1.sol в Remix
2. Скопируйте адрес контракта и ABI
3. Вставьте в script.js
4. Откройте index.html через Live Server
5. Подключите MetaMask (BNB Testnet)
